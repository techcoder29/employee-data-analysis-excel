# Employee Data Analysis Dashboard

## Project Overview

This project presents an end-to-end analysis of employee data using Microsoft Excel. The project focuses on data cleaning, validation, transformation, analysis, and visualization to extract meaningful insights from employee records.

An interactive dashboard was developed to provide a clear overview of workforce distribution and key employee-related metrics.

## Objectives

* Analyze employee data to identify meaningful patterns and trends
* Understand employee distribution across different departments
* Analyze workforce demographics and other employee characteristics
* Perform data cleaning and validation to improve data quality
* Use Pivot Tables and Excel formulas for structured analysis
* Create an interactive and visually informative dashboard

## Tools & Technologies

* Microsoft Excel

* Data Cleaning

## Workbook Structure

The workbook contains the following sheets:

* **Dashboard** – Interactive visual summary of key employee metrics and insights
* **Raw Data** – Original employee dataset
* **Formatted Data** – Structured and formatted version of the dataset
* **Data Cleaning and Validation** – Data quality checks and validation processes
* **Analysis** – Analytical calculations and summarized findings
* **Pivot Tables** – Summary tables used for analysis and dashboard visualizations
* **Insights** – Key findings derived from the analysis
* **Reference Sheet** – Supporting reference information used in the workbook

## Key Features

## Key Insights

The analysis of 1,001 employee records revealed the following key findings:

* **Clothing** has the highest number of employees, with **53 employees**.
* **Outdoors** has the highest average salary among departments, with an average salary of approximately **₹112,055**.
* **Active employees** represent the majority of the workforce, accounting for **85.2%** of all employees (**853 out of 1,001 employees**).
* **Southwest** is the largest region by employee count, with **168 employees**.
* The **USA** has the largest employee population, with **594 employees**.
* The highest individual salary recorded is approximately **₹149,929**, held by a **Director of Sales**.

These findings provide insights into employee distribution, salary patterns, workforce status, and geographical concentration.
